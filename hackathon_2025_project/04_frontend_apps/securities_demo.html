📊 증권서비스 사용자 데이터 조회

카카오페이증권 스타일 UI로 구현된 투자 MBTI 추천 시스템

🚀 주요 기능

🎯
AI 기반 MBTI 추천
사용자의 거래 패턴, 앱 사용 행동, 관심종목을 분석하여 투자 성향에 맞는 MBTI 유형을 자동으로 추천합니다.
📊
실시간 데이터 분석
1,000명의 사용자 데이터를 기반으로 한 현실적인 거래 패턴과 앱 사용 행동 분석을 제공합니다.
📝
설문지 기반 분석
5개 문항의 간단한 설문지를 통해 사용자의 투자 성향을 파악하고 MBTI 유형을 계산합니다.
🔍
상세 분석 리포트
거래 패턴, 앱 사용 행동, 관심종목, 리스크 프로파일, 투자 기간별 상세 분석 결과를 제공합니다.

🎭 투자 MBTI 유형

🔥
불꽃 호랑이 (Growth)
뜨거운 성장주에 올인하는 모험가형
고위험 고수익 • 단기 투자 • 활발한 거래
🦉
든든 올빼미 (Dividend)
배당으로 매달 용돈 받는 안정형
안정적 수익 • 장기 투자 • 보수적 거래
🐢
거북이 플랜 (Index)
ETF 적립으로 느긋하게 장기투자
분산투자 • 매우 장기 • 수동적 투자
🦊
가치 여우 (Value)
숨은 보석 찾아 모으는 저평가 헌터
저평가 종목 • 장기 투자 • 분석적 접근
🐦‍⬛
룰 기반 까마귀 (Quant)
데이터와 규칙으로만 판단하는 이성형
정량적 분석 • 중기 투자 • 체계적 거래
🦌
초록 사슴 (ESG)
환경·사회도 챙기는 착한 투자
지속가능성 • 장기 투자 • 윤리적 투자

📖 사용법

🚀 빠른 시작

1. 더미 데이터 생성 및 API 서버 실행

python run_securities_demo.py

2. React 앱에서 사용

import { SecuritiesDataApp } from './securities_data_integration.js'; function App() { return <SecuritiesDataApp />; }

3. API 직접 호출

# MBTI 자동 추천 curl http://localhost:5000/api/users/user_0001/mbti-recommendation # 설문지 조회 curl http://localhost:5000/api/mbti/questionnaire
