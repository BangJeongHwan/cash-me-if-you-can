📋 사용 가능한 API 엔드포인트

  • GET / - API 서버 정보
  • GET /health - 서버 상태 확인
  • GET /tools - 사용 가능한 도구 목록
  • POST /call - 일반적인 도구 호출
  • POST /joke - 농담 생성
  • POST /ask - OpenAI 질문
  • POST /explain - 개념 설명
  • POST /youtube/search - YouTube 검색
  • POST /youtube/trending - YouTube 인기 동영상
